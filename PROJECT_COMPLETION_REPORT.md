## 🏗️ 技术架构

### 后端技术栈
- **框架**: Spring Boot 2.7.18
- **安全**: Spring Security + JWT
- **数据库**: MySQL 8.0 + MyBatis Plus
- **缓存**: Redis 6.0+
- **服务发现**: 无（直接服务调用）
- **容器化**: Docker + Docker Compose

### 前端技术栈
- **框架**: Vue 3 + TypeScript
- **UI组件**: Element Plus
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由**: Vue Router
- **HTTP客户端**: Axios

### 小程序技术栈
- **框架**: 微信原生小程序
- **开发工具**: 微信开发者工具
- **UI样式**: 原生WXSS + 自定义组件

## 🚀 部署指南

### 测试环境部署
```bash
# 1. 克隆项目
git clone <repository-url>
cd fact2.0

# 2. 执行部署脚本
./scripts/deploy-test.sh

# 3. 访问系统
# 前端: http://localhost
# API: http://localhost/api
```

### 生产环境部署
1. 修改 `docker-compose.prod.yml` 配置
2. 配置域名和SSL证书
3. 执行 `./scripts/deploy-prod.sh`
